# **App Name**: StatementXLS

## Core Features:

- Secure PDF Upload: Allow users to securely upload bank statement PDFs for conversion.
- Automated Data Extraction: Automatically extract data from the PDF bank statement.
- Data Formatting: Format the extracted data into a clean, structured Excel file format.
- One-Click Download: Provide a one-click option to download the converted Excel file.

## Style Guidelines:

- Primary color: Professional blue (#2962FF) to inspire trust and reliability.
- Background color: Light gray (#F5F7FA) for a clean and modern look.
- Accent color: Subtle light blue (#ADD8E6) for interactive elements and highlights.
- Body and headline font: 'Inter' sans-serif font, suitable for both headlines and body text, for a modern, objective feel.
- Use clean and professional icons for navigation and actions.
- Maintain a clear, intuitive layout with easy-to-navigate sections.
- Incorporate subtle animations for loading and file conversion processes.
